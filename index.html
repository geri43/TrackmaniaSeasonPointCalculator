Calculation info

The calculation is based on the tweets on the developers.
There is also a modifier applied for different map groups, which is not yet public,
but I took a guess which seems very close.
Base point calculation:
Rank 1-10: 40,000 / rank
Rank >10: 4000 / (2 ^ (log (rank) - 1))
Speculated tier modifier:
Green: 53/12%
Blue: 53/6%
Red: 53/4%
Black: 53/3%
Final equation: point = base point * (1 - tier modifier / 100)
